Elemento OL, UL, LI

Voltar

O Elemento HTML ol (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados

O elemento HTML ul (ou elemento HTML de Lista desordenada) representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes. O elemento HTML li (ou a Lista dos Itens de um elemento HTML) é usado para representar um item que faz parte de uma lista.

Exemplos:

    Lista ordenada:
  1. Primeiro item ordenado
  2. Segundo item ordenado
  3. Terceiro item ordenado
    Lista desordenada:
  • Item desordenado
  • Item desordenado
  • Item desordenado

Elemento SUP e SUB

Voltar

O elemento HTML sup define um trecho de texto que deve ser exibido, por motivos tipográficos, acima e geralmente menor que o trecho principal do texto, ou seja, em sobrescrito.

O elemento HTML sub define um pedaço de texto que deve ser exibido, por razões tipográficas, abaixo, e geralmente menor, do que o pedaço de texto principal, ou seja, subscrito.
